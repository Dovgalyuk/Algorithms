
    Реализовать в проекте Library двусвязный список (модуль list) и стек на основе этого списка (модуль stack).
    Решить задачу, используя реализованный стек.

Написать программу, работу процессора со стеком. Процессор имеет 4 регистра, обозначенных буквами A, B, C, D. 
Во входном файле присутствуют операции PUSH и POP. Операция PUSH может иметь в качестве параметра число или регистр. 
Операция POP в качестве параметра имеет регистр и считывает в него значение с вершины стека. 
После выполнения последовательности операций программа должна распечатать содержимое регистров.

Входные данные

PUSH 1
POP B

Выходные данные

A = 0
B = 1
C = 0
D = 0

Входные данные

PUSH 1
PUSH 2
POP B
POP C
PUSH C
POP D

Выходные данные

A = 0
B = 2
C = 1
D = 1
