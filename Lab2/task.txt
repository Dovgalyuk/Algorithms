Реализовать в проекте Library односвязный список (модуль list) и стек на основе этого списка (модуль stack).
Решить задачу, используя реализованный стек.
Написать программу, моделирующую работу стека ограниченного объема.
В качестве входных данных программа получает максимально возможный размер стека и последовательность команд:
push N – занести число N в стек, pop – достать число из стека и напечатать его.
Если стек переполняется или производится попытка извлечь значение из пустого стека,
программа должна печатать одно из сообщений: BAD PUSH или BAD POP.
После окончания работы программы, она должна напечатать текущее состояние стека.

Входные данные

2
push 3
push 2
push 4
pop
push 1

Выходные данные

BAD PUSH
2
STACK
3
1
